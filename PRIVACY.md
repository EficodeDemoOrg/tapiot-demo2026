# Privacy Policy

**Last Updated:** 2026-04-09

## Overview

Barbarian Arena is a local-only, client-side HTML5 game. We are committed to protecting user privacy and being transparent about our data practices.

## Data Collection

**This application does not collect, store, transmit, or process any personal data.**

Specifically:
- ❌ No user accounts or authentication
- ❌ No cookies or local storage of personal data
- ❌ No analytics or tracking scripts
- ❌ No network requests to external services
- ❌ No collection of IP addresses, device info, or usage data
- ❌ No personally identifiable information (PII) is handled

## How It Works

- The game runs entirely in the browser using HTML5 Canvas and Web Audio API
- A local Node.js/Express server serves only static files
- All game logic and audio synthesis happen client-side
- The server binds to `127.0.0.1` (localhost only) by default

## Third-Party Services

This application uses no third-party services, APIs, or SDKs that could collect data.

## GDPR Compliance

Since no personal data is collected or processed, GDPR data subject rights (access, rectification, erasure, portability) are not applicable. However, we remain committed to privacy-by-design principles.

## Changes to This Policy

Any changes to this privacy policy will be reflected in this file with an updated date.
